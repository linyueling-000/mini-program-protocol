隐私政策

更新时间：2026-07-10 · 生效时间：2026-07-10

本游戏非常重视您的隐私保护。本隐私政策说明本游戏如何收集、使用、存储和分享您的个人信息。请您仔细阅读本政策，了解您的权利与本游戏对您个人信息的处理方式。

本游戏为单机休闲小游戏，承诺最小化收集原则：仅收集实现游戏核心功能所必需的信息，不收集与服务无关的个人信息。

一、我们收集的信息

信息类型具体内容是否收集用途
微信 OpenID 由微信平台自动分配的用户标识 否 本游戏无登录、无账号体系
位置信息 精确/粗略位置 否 游戏不需要
通讯录 联系人列表 否 游戏不需要
手机号 / 身份证 个人身份信息 否 本游戏不强制实名
设备信息 设备型号 / 系统版本 是 仅用于屏幕适配（getSystemInfoSync）
本地存储 协议同意状态 / 错误日志 是 仅存储在本机，不上传服务器

二、信息的使用方式

本游戏收集的信息仅用于以下目的：

  • 屏幕适配：获取设备屏幕宽高与像素比，用于正确渲染游戏画面（仅在游戏启动时调用一次 wx.getSystemInfoSync）。
  • 本地偏好保存：使用 wx.getStorageSync / setStorageSync 在本机保存协议同意状态、游戏错误日志等纯本地信息，不会上传到任何服务器。
  • 错误日志：在游戏运行异常时，将错误信息保存到本机（wx.storage 'errLog'），便于排查问题，不主动上报。

三、信息的存储与保护

1. 本游戏不设置服务器端，所有数据仅存储在您的设备本地。

2. 本游戏不会将任何信息上传至开发者服务器或第三方服务器。

3. 本游戏不会向任何第三方共享、出售或交换您的个人信息。

4. 您可以通过清除小程序本地缓存彻底删除所有游戏数据。

四、第三方 SDK 清单

本游戏为完全自研，未集成任何第三方 SDK（包括统计 SDK、广告 SDK、推送 SDK、社交分享 SDK 等）。

本游戏仅使用微信小游戏平台官方 wx.* API，微信平台自身的数据收集行为请参阅《微信隐私政策》。

五、未成年人保护

1. 本游戏无内购、无广告、无虚拟货币交易，不涉及任何付费行为，对未成年人无经济风险。

2. 本游戏已接入微信平台适龄提示体系，由微信统一管理未成年人游戏时长；监护人可在微信设置中调整适龄等级或拒绝未成年人使用本游戏。

3. 监护人有权拒绝未成年人继续使用本游戏，可通过设备设置或卸载本游戏实现。

六、您的权利

您对自己的个人信息享有以下权利：

  • 知情权：您有权了解本游戏对您个人信息的处理方式（详见本政策）；
  • 删除权：您可以随时删除本游戏或清除本地缓存，所有数据将被清除；
  • 拒绝权：您可以拒绝同意本政策，停止使用本游戏。

七、政策的更新

本游戏可能根据业务调整、法律法规变化对本政策进行修订。修订后的政策将在游戏内或本页面公布。如您不同意修订后的政策，可以停止使用本游戏。

本游戏承诺：最大程度保护您的隐私安全。不收集与服务无关的信息，不上传本地数据，不集成任何第三方 SDK。
